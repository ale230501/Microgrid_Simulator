# Configs

Canonical YAML configurations live here.

## Controllers
- `configs/controllers/rl/`: RL configs (currently OPSD variants).
- `configs/controllers/mpc/`: MPC configs.
- `configs/controllers/rbc/`: rule-based controller configs.

## Sweeps
- `configs/sweeps/`: ready-to-use sweep example files.

Legacy config paths under `RL_AGENT/PARAMS/`, `MODEL_PREDICTIVE/`, and `RULE_BASED/` are still present for compatibility, but new work should target `configs/`.
