# Tectonic Local Bundle

This folder hosts a local TeX/Tectonic bundle for report generation workflows.

## Contents
- `tectonic.exe`: local executable.
- `tectonic.zip`: local archive (optional backup).
- `cache/`: runtime cache downloaded/generated by Tectonic.

`cache/` and archive artifacts are considered local tooling data and are ignored by git.

If you regenerate or update this bundle, keep scripts referencing `tools/tectonic/tectonic.exe` unchanged to avoid breaking existing paths.
